# Talaria Digital — Web Corporativa

Sitio web estático de [Talaria Digital](https://talariadigital.es), agencia boutique B2B de consultoría, gestión administrativa y desarrollo de negocios digitales.

## Tecnologías

- HTML5 semántico
- [Tailwind CSS](https://tailwindcss.com/) vía CDN
- Google Fonts (Inter)
- Hosting: GitHub Pages con dominio personalizado

## Estructura

```
/
├── index.html          # Landing page principal
├── contacto.html       # Página de contacto (formulario Tally + email)
├── aviso-legal.html    # Aviso legal (LSSICE)
├── privacidad.html     # Política de privacidad (RGPD + LOPDGDD)
├── cookies.html        # Política de cookies
├── condiciones.html    # Términos y condiciones generales
└── assets/
    └── img/            # Logos e imágenes
```

## Despliegue

El sitio se despliega automáticamente en GitHub Pages desde la rama `main`.

Dominio: [talariadigital.es](https://talariadigital.es)
